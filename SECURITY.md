# Security Policy

## Supported versions

Only the latest state of the `main` branch is actively maintained.

## Reporting a vulnerability

If you discover a security vulnerability in this project, please open a private GitHub security advisory or contact the maintainers directly instead of creating a public issue.

We will investigate and respond as soon as possible.

Please include:

- A description of the vulnerability
- Steps to reproduce or locate the issue
- Any suggested fixes if you have them

## Scope

This project fetches publicly available data from the Lost Saga API. Vulnerabilities related to copyright or third-party data should be handled through GitHub issues rather than security reports.
